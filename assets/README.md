# Assets

This folder should contain the following image assets referenced in `app.json`:

- `icon.png` — 1024x1024 app icon
- `splash.png` — splash screen image
- `adaptive-icon.png` — Android adaptive icon foreground (1024x1024)
- `favicon.png` — web favicon (48x48)

Generate these with your brand assets (e.g. a rocket 🚀 icon on a dark `#0a0b0f` background
with the `#6c63ff` accent color), or use `npx expo-asset-generator` / Figma exports.

Recommended palette:
- Background: #0a0b0f
- Accent: #6c63ff
- Accent Light: #8b85ff
